# 📝 更新日志

记录系统的版本更新和功能变化

## 🎉 最新版本

### v2.5.0 (2025-06-24)

#### ✨ 新增功能
- **事件抽取控制开关**：在游戏设置中新增个人事件和团队事件的独立开关
  - 可单独控制个人事件抽取（影响4个事件卡片显示）
  - 可单独控制团队事件抽取（影响团队事件投票）
  - 全部关闭时隐藏所有事件相关界面
- **重抽次数显示控制**：个人事件开关关闭时自动隐藏重抽次数显示
- **docsify文档系统**：全新的在线文档，提供更好的阅读体验
  - 现代化的UI设计和响应式布局
  - 右侧目录导航，方便快速定位
  - 移动端优化，支持触摸操作
  - 智能搜索功能

#### 🔧 改进优化
- **游戏设置界面重构**：采用卡片式布局，提升用户体验
- **开关样式统一**：使用统一的toggle开关设计
- **状态持久化增强**：所有设置自动保存到本地存储
- **错误处理优化**：改进事件抽取的错误提示和异常处理

#### 🐛 问题修复
- 修复了displayRandomMissions函数定义导致的抽取错误
- 修复了事件卡片在页面刷新后异常显示数据的问题
- 优化了多人游戏中的状态同步机制
