# 🚀 快速开始

欢迎来到 UnhappyCar！本指南将帮助您在几分钟内开始您的第一场游戏。

---

## 🎮 单机模式

单机模式是熟悉游戏基本流程和玩法的最佳方式，无需连接服务器。

1.  **进入游戏**
    *   打开游戏页面后，点击 **“进去看看”** 按钮，即可进入单机游戏界面。

2.  **抽取角色**
    *   点击屏幕中央的 **“开始抽取”** 按钮。
    *   系统会为四个玩家位置随机分配角色。

3.  **抽取事件**
    *   角色分配完毕后，点击 **“抽取事件”** 按钮。
    *   系统会为每个玩家分配一个**个人事件**，并在下方展示三个**团体事件**作为投票候选。

4.  **体验游戏**
    *   您可以尝试点击个人事件卡片来消耗**重抽次数**，更换任务。
    *   熟悉角色管理、事件管理等设置项。

---

## 👑 主持多人游戏

当您准备好和朋友一起玩时，可以由一人担任主持人创建房间。

1.  **连接服务器**
    *   打开游戏页面，等待界面提示 **“已连接到服务器”**。

2.  **创建房间**
    *   点击 **“主持游戏”** 按钮。

3.  **邀请朋友**
    *   将房间代码**复制**并发送给您的朋友。

4.  **开始游戏**
    *   等待所有朋友都加入房间后（您可以在界面上看到已加入的玩家），点击 **“开始抽取”**。
    *   作为主持人，您的所有操作（如抽取角色/事件、调整重抽次数）都会**实时同步**给房间内的所有玩家。

> **⚠️重要提示**: 主持人掉线或关闭页面会导致房间解散。为了获得最佳体验，建议使用网络稳定的电脑担任主持人。

---

## 👥 加入多人游戏

作为玩家加入朋友创建的房间非常简单。

1.  **连接服务器**
    *   同样，等待界面提示 **“已连接到服务器”**。

2.  **加入房间**
    *   点击 **“加入游戏”** 按钮。
    *   在弹出的输入框中，输入主持人提供给您的 **房间代码**。

3.  **等待开始**
    *   成功加入后，您将看到游戏主界面。此时，请等待主持人开始游戏。
    *   在游戏中，您可以参与**团体事件投票**，并查看主持人同步过来的所有游戏状态。

> **💡小贴士**: 如果在游戏中遇到界面显示不一致或卡顿的情况，尝试**刷新页面**并重新加入房间，通常可以解决问题。
